# 클룩 예약 QR 코드용 URL 안내

## 메인 앵커 링크들

### 클룩 예약 직접 연결
- **URL**: `https://your-domain.com/#klook-booking`
- **짧은 URL**: `https://your-domain.com/#klook`
- **설명**: 메인 페이지에서 클룩 예약 카드로 바로 이동하여 자동으로 클룩 예약 모드가 활성화됩니다.

### 일반 예약 섹션
- **URL**: `https://your-domain.com/#booking`
- **설명**: 예약 타입 선택 화면으로 이동 (Direct Booking과 Klook Reservation 선택 가능)

## 실제 배포 URL (예시)
```
https://k-recording-cafe.replit.app/#klook-booking
https://k-recording-cafe.replit.app/#klook
https://k-recording-cafe.replit.app/#booking
```

## QR 코드 생성 방법
1. 위 URL을 QR 코드 생성 웹사이트에 입력
2. 권장 사이트: qr-code-generator.com, qr.io 등
3. 클룩 고객이 QR 코드를 스캔하면 바로 클룩 예약 양식으로 이동

## 기능 설명
- `#klook-booking` 또는 `#klook` 해시가 있는 URL로 접속 시
- 자동으로 "Klook Reservation" 모드가 활성화됨
- 클룩 예약 ID 입력 필드가 나타남
- 음료 선택 및 추가 서비스 선택 가능
- 즉시 예약 완료 (결제 없음)

## 사용 시나리오
1. **클룩에서 예약 완료한 고객**
2. **QR 코드 스캔** → 클룩 예약 양식 자동 표시
3. **클룩 예약 ID 입력**
4. **음료 및 추가 서비스 선택**
5. **예약 완료**
